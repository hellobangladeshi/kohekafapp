# 🍔 Kohekaf Food Delivery

A complete professional food delivery web application with Firebase backend. This application supports multiple user types including Customers, Delivery Drivers, Restaurant Owners, and Admins with real-time order tracking and location services.

## ✨ Features

### 👤 Customer Features
- User registration and login
- Browse restaurants and food items
- Search and filter by category
- Add items to cart
- Place orders with delivery location
- Real-time order tracking
- View order history

### 🚗 Driver Features
- Online/offline status toggle
- Receive real-time order notifications
- Accept/reject delivery requests
- Update delivery status
- View earnings and delivery history

### 🏪 Restaurant Features
- Manage food menu (add/edit/remove items)
- View incoming orders
- Update order status (confirmed → preparing → ready)
- Track order history

### 👑 Admin Features
- Dashboard with analytics
- Manage all users
- Monitor orders
- Manage restaurants
- View revenue statistics

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Firebase (Authentication, Firestore Database, Storage)
- **Maps**: Google Maps API
- **Deployment**: Firebase Hosting / GitHub Pages
